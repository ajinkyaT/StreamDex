# Streamdex Brand Direction

## Core theme

Signal over noise.

The brand should feel like calm control over chaotic information.
It is not generic AI magic.
It is not an algorithmic feed.
It is a personal video stream built from trusted sources.

## Visual story

Move from:
- cluttered feeds
- dense text
- notification pressure
- algorithmic overwhelm

Into:
- clean motion
- trusted sources
- concise video cards
- calm focus
- personal control

## Palette

- Midnight Navy: `#07131F`
- Deep Slate: `#13283A`
- Signal Cyan: `#16D8FF`
- Mist White: `#EAF6FF`
- Alert Coral: `#FF6B5E`
- Evidence Amber: `#FFBE55`

## Usage

- Use Midnight Navy as the main canvas.
- Use Signal Cyan for active states, logo glow, stream lines, and key moments.
- Use Mist White for typography and captions.
- Use Alert Coral only in the opening problem section.
- Use Evidence Amber for source chips, citations, and proof markers.

## Recommended split

- 70% Midnight Navy and dark surfaces
- 15% Mist White
- 10% Signal Cyan
- 5% Coral and Amber combined

## Typography

Recommended:
- Headlines: Space Grotesk
- Supporting text and UI: Manrope

Fallback:
- Headlines: Sora
- Supporting text and UI: Manrope

## Logo guidance

The logo should communicate:
- video
- connected sources
- personal stream
- curated signal

The triangle represents video.
The orbit and nodes represent connected tools and sources.
The inward focus represents curation and relevance.

## Motion language

Use:
- dots converging into a stream
- source cards flowing into a central player
- smooth horizontal movement
- soft pulses for trusted sources
- text blocks collapsing into video

Avoid:
- noisy glitch effects
- crypto-style neon overload
- purple gradients
- random particle explosions
- overly playful motion

## Closing mood

The ending should feel earned, calm, and product-grade.
The viewer should feel that Streamdex is how they reclaim attention from algorithmic feeds.
